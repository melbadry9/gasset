## Installation

- `git clone https://github.com/melbadry9/gasset.git`
- `sudo apt-get install python3 python3-pip`
- `pip3 -r install requirements.txt`
- Add your facebook cookies into `cookie.py`

## Usage

`python3 asset.py domain.com`
